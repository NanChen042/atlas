# Atlas

# Atlas · Personal Knowledge & Productivity System

> 一个 **Local-first、AI-augmented** 的个人知识与生产力系统  
> 用工程化方式，构建长期可演化的「第二大脑」

---

## 📌 项目简介

Atlas 是一个面向个人的 **知识与生产力系统**，目标不是记录信息，而是：

> **将碎片化信息转化为可理解的认知，并进一步转化为可执行的行动。**

它关注三个核心问题：

- 知识如何被**高效存储与组织**
- 思考过程如何被**结构化与复用**
- 行动如何从知识中**自然产生**

这是一个偏工程系统的 Web App，而不是传统意义上的笔记工具或 Todo 应用。

---

## 🎯 设计目标

- **Local-first**  
  数据优先保存在本地，离线可用、性能稳定、隐私可控

- **AI as Accelerator（而非主导）**  
  AI 用于总结、关联、检索，而不是替代思考

- **系统化而非功能堆叠**  
  关注信息流转与状态演进，而非堆功能

- **长期可演化**  
  支持持续扩展，而不是一次性 Demo

---

## 🧠 核心理念

### 1. 知识不是“存下来”，而是“能被调用”

没有上下文、无法复用的内容，只是信息噪声。

### 2. 生产力来自于“转化效率”

从信息 → 认知 → 决策 → 行动，链路越短，系统越有价值。

### 3. AI 永远不做最终决策

AI 是工具，系统的控制权始终在人。

## 🧩 核心模块（概览）

### Knowledge System ✅

- Markdown 编辑与预览
- 双向链接（`[[关键词]]` 语法）
- 标签系统与状态筛选
- 知识状态管理（草稿 / 已理解 / 可行动）
- AI 总结、续写、润色、标签生成
- OCR 图片文字识别

### Productivity System ✅

- 从知识笔记一键派生任务
- 任务优先级与截止日期管理
- 看板视图（Kanban Board）
- 习惯追踪与连续打卡统计
- 番茄钟专注计时

### Dashboard ✅

- 今日任务与习惯汇总
- 本周专注时长图表
- 连续活跃天数统计
- 快速操作入口

### AI Assistant ✅

- 多模型聊天对话（硅基流动 / DeepSeek / 通义千问）
- 笔记内容 AI 总结、续写、润色
- AI 自动生成标签
- DeepSeek-OCR 图片文字识别
- 可配置、可关闭、可替换

### Local-first Storage ✅

- IndexedDB 持久化存储（idb 库）
- 数据导出 / 导入（JSON 格式）
- 离线优先，所有数据保存在本地

## 🛠 技术栈（当前规划）

- **Framework**: Vue 3
- **Build Tool**: Vite + Bun
- **State Management**: Pinia
- **Editor**: TipTap / ProseMirror
- **Storage**: IndexedDB / OPFS
- **Compute**: Web Worker
- **Styling**: Tailwind CSS
- **AI**: 可插拔 API（OpenAI / 本地模型）

> 技术选型以 **可维护性、长期演进能力** 为优先，而非追逐新奇。
